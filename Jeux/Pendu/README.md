# Jeu du pendu

## Description

Ce programme permet de jouer une partie au jeu du pendu dans la console.

![pendu](https://github.com/user-attachments/assets/4607c8e8-2e68-4c50-b182-1a53add889d0)

## Gradle

### Build

<pre>
gradle build
</pre>

### Run

<pre>
gradle run --args="src/main/resources/dico.txt" --console=plain
</pre>
