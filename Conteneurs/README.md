# Exercices de base

Ce dépôt contient un ensemble d'exercices concernant les structures de données. L'implémentation est réalisée en langage C.

Les structures en question sont :

* Les piles
* Les files
* Les arbres
* Les tables de hachage

Les codes présentés ont valeur d'exercices et non d'implémentations de référence, l'objectif étant d'introduire ces structures
en les programmant.
