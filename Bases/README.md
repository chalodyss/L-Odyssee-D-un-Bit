# Exercices de base

Ce dépôt contient pour un ensemble de langages un fichier contenant l'implémentation de 15 fonctions servant d'exercices
d'introduction pour se familiariser avec le langage en question. Les exercices sont accessibles et de difficulté variable.

Les fonctions à écrire sont :

* Le traditionnel "hello world"
* L'affichage des nombres de 0 à n
* L'affichage des lettres de l'alphabet
* Le comptage de voyelles dans un mot
* Le comptage de consonnes dans un mot
* Le palindrome
* La valeur absolue d'un entier
* La primalité d'un entier
* Savoir si un entier est parfait
* La conversion binaire d'un entier
* La conversion de base allant de 2 à 16
* Le chiffrement de César
* Le tri à bulles
* Le nombre d'occurrences dans un tableau
* La notation postfixée

Gardez à l'esprit que les implémentations fournies ici sont des exemples et que l'on peut les écrire parfois de
manière différente.
