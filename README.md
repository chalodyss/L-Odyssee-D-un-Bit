# L'Odyssée d'un Bit

## Description

Ce dépôt contient une partie des programmes que j'ai écrits dans l'objectif d'approfondir mes connaissances sur les ordinateurs,
notamment par la programmation.

L'objectif est de présenter un ensemble de codes sources couvrant des thèmes variés permettant la construction étape par étape
d'une base technique en programmation ainsi que d'une culture informatique préalable et indispensable à l'étude des ordinateurs
au sens large du terme.

Les sources présentées constituent une base de travail pédagogique et ne sont donc pas des projets finalisés ou optimisés.

Le dépôt sera mis à jour en fonction de mes disponibilités.
