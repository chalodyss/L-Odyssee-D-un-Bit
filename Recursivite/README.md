# Récursivité

Ce dépôt contient un ensemble de fonctions écrites de deux manières différentes : itérative et récursive. L'objectif ici
est de travailler la notion de récursivité en passant d'une forme à une autre lors de l'écriture d'une fonction.

Les fonctions traitées sont :

* L'affichage des entiers de 0 à n
* La factorielle
* La suite de Fibonacci
* Les tours de Hanoï
* La fonction d'Ackermann

Ces exercices peuvent contenir de la difficulté mais sont très instructifs.
